## Section 1 — General Awareness

### 1. What is web development, and why is it essential in the context of the internet?

Web development is the process of building websites and web applications. It is
important because websites are used every day for communication, business, and
services.

### 2. What does HTML stand for, and what is its primary purpose?

Yes. HTML stands for **HyperText Markup Language**. It is used to create the
structure of a web page.

---

## Section 2 — Familiarity with CSS

### 3. What is CSS used for in web development?

Yes. CSS is used to style websites — such as colors, fonts, spacing, and layouts.

---

## Section 3 — JavaScript and Its Importance

### 4. What is JavaScript, and why is it crucial for interactive websites?

Yes. JavaScript is a programming language that adds interactivity to websites,
like forms, buttons, and dynamic content.

---

## Section 4 — Introduction to React and Node.js

### 5. What are React and Node.js commonly used for?

Yes.

- **React** is used to build modern and interactive user interfaces.
- **Node.js** is used for backend development and creating APIs.
